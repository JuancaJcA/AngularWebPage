RESUMEN DE LA MATERIA

En la actualidad toda institución requiere interactuar de menor a mayor manera con las nuevas tecnologías de información y comunicación siendo el uso de las tecnologías online
una herramienta primordial para incorporar a la institución en el nuevo enfoque de mercado laboral y competitividad empresarial. El buen manejo de las tecnologías
online proporciona una ventaja competitiva a los profesionales e instituciones que hacen uso efectivo de estas tecnologías.

Algunas de las habilidades que aprenderás son:

Diseño Responsive

Se darán los fundamentos de HTML y CSS para que puedas realizar páginas web responsive.

Pasión

Se comprenderán diferentes términos de vital importancia para lograr comprender las páginas web de una manera apasionada.

Diseño

Se tendrá un énfasis especial respecto a los diseños que realices.

Soporte

Tendrás a tu disposición a un docente de tiempo completo para que puedas aprender cada día más y más, despejando cualquier duda.

TEMAS

Los temas que se llevarán en el semestre seŕan los siguientes:

Introducción a la Tecnología WEB

Se comprenderán diferentes conceptos web, para que se pueda comprender de mejor manera en el desarrollo, para que siempre se tenga una base teórica.

Solución tecnológica WEB

Se llevarán los fundamentos de GitHub, Bitbucket, SOA, SaaS, DOM, DHTML y mucho más! De la misma manera se realizarán ejemplos prácticos.

CMS

Se realizará una explicación de los mejores CMS hoy en día, de la misma manera se planteará un proyecto para que el mejor de la materia represente al curso.

Frameworks

Se revisarán los frameworks más usados hoy en día, como VUE o Angular. De la misma manera se plantearán proyectos para revisar si hubo una correcta comprensión.
